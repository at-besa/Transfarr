# Transfarr - Future Features List

This document keeps track of planned and proposed features for future development phases.

## 1. Desktop Application & System-Tray Integration
- Evaluate and integrate a desktop framework (e.g., **MAUI Blazor Hybrid**, **Photino**, or **WPF/WebView2**) to run the Node application natively on the desktop.
- Implement **System-Tray logic** (minimize to tray, background service functionality).
- Create a professional **Installer** (e.g., InnoSetup or WiX Toolset) for easy distribution to end-users.

## 2. Reputation System Expansion
- Expand the rudimentary point system (10 GB = 1 point).
- Introduce user ranks (e.g., Bronze, Silver, Gold) or privileges (e.g., higher download priority from peers you have uploaded heavily to).

## 3. Application-Integrated Update System
- Implement an automated update check on startup.
- Allow users to download and install updates directly from the application UI.
- Use a dedicated update-hub or integrate with the Global Hub for update distribution.

## 4. Advanced Search Filters & Sorting
- Implement category-based filtering (e.g. Movies, Music, Software).
- Add sorting options by size, peer count (swarm health), and modification date.
- Support "Search within results" functionality.

## 5. Peer-to-Peer Group Chats
- Allow users to create and join persistent chat rooms/channels on the Hub.
- Implement basic moderation tools for channel owners.
